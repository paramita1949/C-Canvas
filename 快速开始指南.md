# Canvas Cast - 滚动优化快速开始指南

## 🚀 5分钟上手

### 1. 编译项目（1分钟）
```bash
dotnet build --configuration Release
```

### 2. 启动应用（10秒）
- 运行编译后的可执行文件
- 观察Console窗口的GPU状态信息

### 3. 导入测试图片（30秒）
- 点击 "📁 导入" 按钮
- 选择一张4K图片

### 4. 体验新缓动函数（3分钟）

#### 试试超级平滑 ⭐
1. 在图片区域右键
2. 滚动函数 → **⭐ 超级平滑**
3. 点击图片底部
4. 感受丝滑的滚动体验！

#### 对比原有贝塞尔
1. 右键 → 滚动函数 → **贝塞尔曲线**
2. 再次点击图片底部
3. 对比感受差异

#### 其他有趣选项
- **🎯 物理模拟** - 像真实世界的滚动
- **🌊 弹性平滑** - 到达终点时有轻微回弹
- **🧠 智能自适应** - 自动根据距离优化

---
