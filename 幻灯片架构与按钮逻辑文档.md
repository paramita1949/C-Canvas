## 1. 总体架构概述

### 1.1 应用程序结构
咏慕投影是一个基于WPF的幻灯片演示应用程序，采用**数据库优先**的架构模式，使用Entity Framework Core + SQLite进行数据存储。

### 1.2 核心架构层次
```
┌─────────────────────────────────────────────┐
│              UI层 (WPF)                     │
│  ┌─────────────┬─────────────┬─────────────┐ │
│  │ MainWindow  │ Controls    │ Settings    │ │
│  └─────────────┴─────────────┴─────────────┘ │
├─────────────────────────────────────────────┤
│             业务逻辑层 (Managers)           │
│  ┌─────────────┬─────────────┬─────────────┐ │
│  │TextProject  │Projection   │ImageProcess │ │
│  │Manager      │Manager      │Manager      │ │
│  └─────────────┴─────────────┴─────────────┘ │
├─────────────────────────────────────────────┤
│              数据模型层 (Models)            │
│  ┌─────────────┬─────────────┬─────────────┐ │
│  │TextProject  │Slide        │TextElement  │ │
│  └─────────────┴─────────────┴─────────────┘ │
├─────────────────────────────────────────────┤
│           数据访问层 (Entity Framework)      │
│  ┌─────────────────────────────────────────┐ │
│  │         CanvasDbContext (SQLite)        │ │
```

3. **主窗口初始化** → MainWindow构造函数
4. **数据库初始化** → CanvasDbContext

## 2. 核心数据模型

### 2.1 TextProject (文本项目)
```csharp
位置: Database/Models/TextProject.cs
用途: 表示一个完整的幻灯片演示项目
主要属性:
- Id: 项目唯一标识符
- Name: 项目名称
- Width: 画布宽度(默认1920)
- Height: 画布高度(默认1080)
- BackgroundImage: 项目背景图片
- CreatedAt: 创建时间
- Slides: 包含的幻灯片集合
```

### 2.2 Slide (幻灯片)
```csharp
位置: Database/Models/Slide.cs
用途: 表示演示文稿中的单个幻灯片页面
主要属性:
- Id: 幻灯片唯一标识符
- TextProjectId: 所属项目ID
- Title: 幻灯片标题
- SortOrder: 显示顺序
- BackgroundColor: 背景颜色
- BackgroundImage: 背景图片
- SplitMode: 分屏模式(单页/分屏)
- TextElements: 文本元素集合
```

### 2.3 TextElement (文本元素)
```csharp
位置: Database/Models/TextElement.cs
用途: 表示幻灯片上可编辑的文本框
主要属性:
- Id: 文本元素唯一标识符
- SlideId: 所属幻灯片ID
- X, Y: 位置坐标
- Width, Height: 尺寸大小
- ZIndex: 显示层级
- Content: 文本内容
- FontSize, FontFamily: 字体设置
- TextColor, BackgroundColor: 颜色设置
- BorderSettings: 边框设置
- ShadowSettings: 阴影设置
- RichTextSpans: 富文本片段集合
```

### 3.2 核心UI控件

### 3.3 UI模式切换
应用程序支持三种工作模式：

1. **文件模式 (BtnShowFiles_Click)**
   - 用于导入和管理媒体文件
   - 幻灯片编辑功能禁用

2. **幻灯片模式 (BtnShowProjects_Click)**
   - 主要的幻灯片编辑模式
   - 所有编辑功能可用

3. **圣经模式**
   - 圣经文本显示和编辑
   - 特定的圣经相关控件可用

## 5. 按钮逻辑流程

### 5.1 标准按钮事件处理模式
```csharp
private async void Btn[Action]_Click(object sender, RoutedEventArgs e)
{
    // 第1步: 状态验证
    if (_currentTextProject == null) return;
    if (_currentSlide == null) return;

    // 第2步: 用户确认(破坏性操作)
    if (isDestructive)
    {
        var result = WpfMessageBox.Show("确认操作?", "警告",
            MessageBoxButton.YesNo, MessageBoxImage.Warning);
        if (result != MessageBoxResult.Yes) return;
    }

    // 第3步: 业务逻辑执行
    await ExecuteBusinessLogic();

    // 第4步: 数据持久化
    await _dbContext.SaveChangesAsync();

    // 第5步: UI状态更新
    UpdateUIState();

    // 第6步: UI刷新
    LoadSlideList();
}
```

### 5.2 具体按钮逻辑流程示例

#### 5.2.1 新建幻灯片流程
```
BtnAddSlide_Click
    ↓
检查当前项目是否存在 (_currentTextProject)
    ↓
查询当前幻灯片数量和最大排序值
    ↓
创建新的Slide对象:
    - Title: "幻灯片 {count + 1}"
    - SortOrder: maxOrder + 1
    - BackgroundColor: 默认黑色
    - SplitMode: SplitMode.Single
    ↓
添加到DbContext.Slides
    ↓
保存到数据库
    ↓
刷新幻灯片列表
    ↓
选中新创建的幻灯片
```

#### 5.2.2 添加文本框流程
```
BtnAddText_Click
    ↓
验证项目和幻灯片存在
    ↓
创建TextElement对象:
    - Position: (100, 100)
    - Size: (600, 200)
    - ZIndex: 最大值(始终在最上层)
    - FontFamily: 从FontFamilySelector获取当前字体
    - FontSize: 36
    ↓
添加到DbContext.TextElements
    ↓
保存到数据库
    ↓
创建SelectableTextBox UI控件
    ↓
添加到画布
    ↓
自动选中新创建的文本框
    ↓
显示浮动工具栏
```

#### 5.2.3 文本格式化流程
```
BtnBold_Click (斜体、下划线类似)
    ↓
检查是否有选中文本框 (_selectedTextBox)
    ↓
检查是否有文本内容被选中
    ↓
获取选中文本的当前格式状态
    ↓
应用/取消粗体格式:
    - 如果当前粗体 -> 取消粗体
    - 如果当前非粗体 -> 应用粗体
    ↓
更新按钮状态 (视觉反馈)
    ↓
标记内容已修改 (MarkContentAsModified)
    ↓
如果投影未锁定，更新投影显示
```

#### 5.2.4 背景图片导入流程
```
BtnBackgroundImage_Click
    ↓
显示上下文菜单 (单图/多图导入)
    ↓
ImportSingleImageAsSlide 或 ImportMultipleImagesAsSlidesAsync
    ↓
文件选择对话框
    ↓
图片处理 (ImageProcessor.ProcessImageAsync):
    - 缩放到适合尺寸
    - 生成缩略图
    ↓
为每张图片创建Slide对象:
    - BackgroundImage: 处理后的图片路径
    - ThumbnailPath: 缩略图路径
    ↓
保存到数据库
    ↓
刷新幻灯片列表
    ↓
选中新导入的幻灯片
```

## 6. 状态管理机制

### 6.1 模式状态管理
```csharp
// 三种主要模式切换
private enum WorkMode
{
    File,      // 文件管理模式
    Slide,     // 幻灯片编辑模式
    Bible      // 圣经显示模式
}

// 模式切换时的UI状态更新
private void UpdateViewModeButtons(WorkMode mode)
{
    switch (mode)
    {
        case WorkMode.File:
            // 启用文件相关按钮，禁用幻灯片编辑按钮
            BtnImport.IsEnabled = true;
            BtnAddSlide.IsEnabled = false;
            BtnAddText.IsEnabled = false;
            break;
        case WorkMode.Slide:
            // 启用幻灯片编辑按钮，禁用文件管理按钮
            BtnImport.IsEnabled = false;
            BtnAddSlide.IsEnabled = true;
            BtnAddText.IsEnabled = true;
            break;
        case WorkMode.Bible:
            // 启用圣经相关按钮，禁用其他功能
            // 特定处理逻辑
            break;
    }
}
```

### 6.2 文本选择状态管理
```csharp
private SelectableTextBox _selectedTextBox;

// 文本选择状态变化时的按钮状态更新
private void UpdateToolbarButtonStatesFromSelection()
{
    if (_selectedTextBox == null)
    {
        // 禁用所有格式化按钮
        SetFormattingButtonsEnabled(false);
        return;
    }

    // 根据选中文本的格式状态更新按钮
    UpdateBoldButtonState(_selectedTextBox.IsSelectionBold());
    UpdateItalicButtonState(_selectedTextBox.IsSelectionItalic());
    UpdateUnderlineButtonState(_selectedTextBox.IsSelectionUnderline());

    // 更新对齐按钮
    UpdateAlignmentButtons(_selectedTextBox.TextAlignment);
}
```

### 6.3 投影状态管理
```csharp
// 投影锁定状态
private bool _isProjectionLocked = false;

private void BtnLockProjection_Click(object sender, RoutedEventArgs e)
{
    _isProjectionLocked = !_isProjectionLocked;

    if (_isProjectionLocked)
    {
        // 锁定状态: 幻灯片更改不影响投影
        BtnLockProjection.Background = Brushes.Orange;
        BtnLockProjection.Content = "🔒 解锁投影";
    }
    else
    {
        // 解锁状态: 幻灯片更改实时更新投影
        BtnLockProjection.Background = Brushes.LightGray;
        BtnLockProjection.Content = "🔓 锁定投影";
    }
}
```

```csharp
await _dbContext.SaveChangesAsync();

LoadSlideList();

if (!_isProjectionLocked)

    await _projectionManager.UpdateProjection(_currentSlide);
```

### 8.3 UI刷新模式
```csharp
// 完整刷新
private void LoadSlideList()
{
    SlideList.Items.Clear();
    foreach (var slide in _currentTextProject.Slides.OrderBy(s => s.SortOrder))
    {
        var item = CreateSlideThumbnailItem(slide);
        SlideList.Items.Add(item);
    }
}

// 选择性更新
private void UpdateSlideItem(Slide slide)
{
    var item = FindSlideItem(slide.Id);
    if (item != null)
    {
        UpdateThumbnail(item, slide);
    }
}
```

## 9. 错误处理与用户反馈

### 9.1 验证模式
```csharp
// 操作前验证
private bool ValidateOperationContext()
{
    if (_currentTextProject == null)
    {
        WpfMessageBox.Show("请先打开或创建项目");
        return false;
    }

    if (_currentSlide == null)
    {
        WpfMessageBox.Show("请先选择一个幻灯片");
        return false;
    }

    return true;
}

// 文本选择验证
private bool ValidateTextSelection()
{
    if (_selectedTextBox == null)
    {
        WpfMessageBox.Show("请先选中文本框");
        return false;
    }

    if (string.IsNullOrEmpty(_selectedTextBox.SelectedText))
    {
        WpfMessageBox.Show("请先选中要格式化的文本");
        return false;
    }

    return true;
}
```

### 9.2 用户反馈机制
- **破坏性操作确认**: 删除、清除等操作显示确认对话框
- **进度指示器**: 异步操作显示进度条
- **错误消息**: 具体失败原因的提示信息
- **状态提示**: 操作成功/失败的Toast通知

### 9.3 异常处理模式
```csharp
try
{
    // 业务逻辑
```
