# Changelog

## 6.0.5.4 (2026-05-06)

### Added

- 新增独立的“字体与字号”控制中心，集中调整文件夹、文件、标签与菜单字号。
- 新增字号滑条、加减按钮和手动输入，统一支持 0.5 步长调节。
- 新增字号数值归整与范围限制逻辑，并补充半点步长与边界测试。
- 文本项目右键菜单新增“导入幻灯片项目”入口，便于从项目列表直接导入。

### Changed

- 文件树右键“字号设置”由多级字号菜单改为单入口控制中心，减少菜单层级。
- 控制中心接入主题资源，跟随现有配色系统刷新强调色、边框、文字和卡片背景。

## 6.0.5.3 (2026-05-05)

### Added

- 新增仓库级构建清理规则：自动清理 `obj/bin` 下 `*_wpftmp*` 与 `.tmp` 临时目录，减少编译残留。

### Changed

- NDI 启动策略调整：仅 `NDI总开关` 重启后默认关闭，具体通道功能配置保持持久化。
- 歌词 NDI 路由修正：主投影通道与透明通道发布链路拆分，避免普通投影误走透明层逻辑。
- 歌词 NDI 帧发布接口升级：支持主投影帧与透明帧分离传输，修复图层上下关系错乱问题。
- 移除 NDI 调试日志输出与 `ndi-debug.log` 文件落盘，控制台刷新提示不再显示日志路径。

## 6.0.5.2 (2026-05-03)

### Added

- 新增 `NDI网络流控制台` 独立窗口（`文件 -> NDI网络流` 入口打开），集中管理总开关、歌词通道、字幕通道、歌词透明、水印设置与连接状态。
- 新增 `帧水印设置` 独立窗口，支持文字、位置、字号、透明度、字体统一配置，并与 NDI 控制台联动调用。

### Changed

- 移除顶部菜单栏 NDI 快捷按钮，NDI 入口统一回归 `文件 -> NDI网络流`。
- NDI 控制台改为卡片化信息架构（总览指标 + 控制区 + 状态工具区），优化状态反馈与交互层级。
- NDI 控制台交互文案统一为“NDI输出”语义，避免与 AI 字幕识别开关混淆。
- NDI 控制台歌词选项重构为父子关系：`启用歌词 NDI 输出` 下挂 `启用歌词透明（NDI）`，并增加“已生效/待生效/未启用”状态提示。
- NDI 控制台布局优化：提高窗口高度并增加中部滚动容器，修复高 DPI 与多条目场景下遮挡问题。
- 帧水印设置窗口移除字号预览区与动态缩放逻辑，避免大字号场景抖动和裁切，保留核心配置项（文字、位置、字号、透明度、字体）。
- NDI 控制台标题区改为“状态徽章 + 通道卡片刷新按钮”布局，移除顶部突兀刷新块，统一胶囊化视觉风格。
- NDI 控制台窗口高度与栅格间距进一步压缩（`720 -> 680`），减少纵向空洞并避免右上区文字挤压。
- NDI 控制台右侧“水印（实时）”卡片完成等高与基线对齐微调：统一标签间距、输入框高度、数值框宽度与控件纵向节奏，提升整体工整度。
